##Enveronment

For use this system:
1. Create file named '.env'
2. Put into this your app key and secret for social's applications
for example:
FB_key='your Facebook application key'
FB_secret='your Facebook application secret'

Noew we use:
FB == Facebook
VK == VKontakte
TW == Twitter
LI == LinkedIn
